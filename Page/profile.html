    アニメ(サウンドトラック)

    好きなアニメを語っていると永遠に終わらなくなるのでここでは違う方面のお話を書くことにする.

    「サウンドトラック」や「BGM」と言えば,世間的にはやっぱりゲームを思い浮かべることが多いだろう. スーパーマリオブラザーズ初代の地上ステージBGM,星のカービィの「グリーングリーンズ」, ポケモンの戦闘BGMなど...もちろんゲームのサントラも好きで聴くことは多い.

    個人的にはアニメのサントラやBGMが好きである. 有名どころで言えばサザエさんやこち亀のBGM.
    サザエさんは2013年にサントラ「サザエさん音楽大全」がリリースされておりみんなが聴いたことある曲が たくさん収録されている.
    これはゲームのサントラにも当てはまるだろうけど面白いのがどのBGMにも曲名があること. 「楽しい磯野家」や「サザエのテーマ2」などこれだけだと全然何が何なのかわけわかめだけど... 聴けば知らない人はいないくらい馴染みのある曲である. 有名どころであれば逆に曲を聴いて何のアニメかわかるのも面白い.

    こち亀も有名なBGMがある.「コミカルに追いかけっこ1」である. この曲を聴けば現実生活で真面目にやばい失敗をした時,誰でもどうでもよくなってしまう魔法のBGMである.←違う
    日常生活でも役に立ちそうなBGMはたくさん存在しそうだ!!

    個人的に一番好きなサントラは,「ココロ図書館」というアニメのサントラ. このサントラに収録されているBGMはもはやアニメ抜きにして素敵な音楽だと思う. ほぼ全てのBGMが優しい雰囲気で心が温まる.日常生活で癒しで定期的に聴いている,聴きたくなる. 今日,リラクゼーションミュージック(睡眠,ヒーリングミュージックなど)なるものがようつべなどでたくさん投稿されている. 感覚的にはこれに近いのかもしれない.
    アニメ自体は2002年と相当古い作品.作風はいわゆる日常系,癒し系で心が温まるストーリーである. アニメを見た後にこのサントラを聴いて思い出に浸ることもできるし,これ単体でもとても癒される.

    「ココロ図書館」のサントラ...オヌヌメです(。-`ω-)
    (ちなみに昔のアニメだけど「ココロ図書館」は今でもdアニメストアなどで見れます.(2026/5月現在) )

    ゲーム(Minecraft)

    マイクラ歴13年. 当時PS3版(PlayStation 3 Edition)発売時に初めて触ってドハマりした. 当時はマップの広さも有限で地下もそんなに深くなくとにかく狭かった印象. 2018年ぐらいになって個人的なブームは一旦落ち着いた.
    今日,再びMinecraft(マインクラフト)にハマっている. アップデートが繰り返されて今ではとてもボリューミーなゲームである.
    そしてこのゲーム...何と終わりがない. 冒険しても良し,建築しても良し,スローライフを送るも良しの何でもあり. 時間がわけわかめのように溶けまくる.ハマるといくらでもできちゃう.

    しかし,こういう系のゲームは人によって好き嫌いが分かれると思う.自由度が高すぎるのも それはそれで大変.(一応「エンドラ討伐」という目的があるが...)
    それのせいなのか周りでマイクラをやってる人が1人もいない!!これは悲しいことだ. つまりマルチ未経験である()
    だれかフレンドになってください!!一緒にやりましょう!!(切実)

    ゲーム(アイドルマスター)

    ゲームのジャンルに入ってるけども最初はニコニコ動画でアイマスを知った. 「やよぴったん」「てってってー」「とかち」などのネタ(?)動画が
    自分のPのスタートだった.その時はちょっと覗いたくらいだったと思う.

    その後,アニメを見てドハマりした.「THE IDOLM@STER」と「アイドルマスター シンデレラガールズ」. アイマスはみんなが主役でストーリー構成はとても難しそうだけど,この2つはとても上手くまとまっていた. アイマスに詳しくない人でも見やすいアニメだったと思う.

    そしてふとした時ゲームをやってみようと思った.今はサ終したGREEの「アイドルマスター ミリオンライブ！」(通称グリマス). 基本ポチポチしていくだけの作業ゲーみたいなシステムだけども意外と面白い.気軽にやれたのは 導入としてとても良かった.
    その後,「アイドルマスター シンデレラガールズ スターライトステージ」(通称デレステ)が リリースされた.当時3DCGのライブが背景の音ゲーのスマホアプリはかなり珍しかった希ガス. モデルがぬるぬる動いてシーンの演出も綺麗でとても感動した記憶がある. ここからずっとゲームにハマっていく. エンジョイ勢に変わりはないけどどのイベントも報酬が貰えるくらいちゃんとやってた.

    その後グリマスの後継作「アイドルマスター ミリオンライブ! シアターデイズ」(通称ミリシタ)もリリースされた. この2つを軸に10年くらいアイマスのアプリゲーをやっている. グリマスと比べて多少ハードルは上がるけども操作方法も簡単で知ってる曲もたくさんあってとても楽しくプレイしていた.
    その他「アイドルマスター2」,「アイドルマスター ワンフォーオール」,「アイドルマスター シャイニーカラーズ」 などもプレイした.初代アーケードの「アイドルマスター」もプレイした.

    今では一昔前と比べてかなりエンジョイ勢になってると思う(デレステは既に新規要素の追加が終わっていてそういうのも理由にありそう). これからもまったりとアイマスに触れていこうと思う.

    ゲーム(BVE)

    名前の通りである.えっ?何の略かだって...?ということで語源は「房総ビューエクスプレス」. JR東日本255系の愛称からきているってことだね.
    つまりBVEは鉄道運転のシミュレーションゲーム(正式名称はBVE Trainsim). 今は鉄っちゃんではないけど当時の血が騒ぐ. やっぱり今も何だかんだで鉄道が好きなのかもしれないね...

    コンピュータ(CG)

    小さい頃からPCでカチャカチャやってた.最初に触ったOSはWindows XP. Flash黄金時代で「恋のマイアヒ」や「ウマウマ」などのネタ動画を見たり「スーパーノコノコブラザーズ」や マ◯オのパクリゲーをよくやっていた.Yahoo! きっずの「ポケモンガーデン」も懐かしい.

    Windows Vistaの時代では,3D鉄道シミュレータのRailSimをよくやっていた. 3D空間上に鉄道,線路や建物などを配置して鉄道を動かしいわゆるPC上で行うジオラマみたいなゲームである. これが初めて3DCGの楽しさを実感したゲームだったと思う. 自分も当時,建物を作ってみたいと思ってMetasequoiaをDLして作ったことがあったが完成度は酷かった記憶. しかも当時は今ほどモデリングソフトは普及してなかったと思うし資料も少なかった気がしてかなり大変だったような...
    今はCGソフトと言えばBlenderが主流だと思うけど情報はいくらでも転がっててかなりCGの敷居は下がったな～と感じる.

    CGに真剣に興味を持ったのは高校生ぐらいの時だと思う. 学んだ数学や物理学とCGとの関係性を知るのが楽しかった. 何の記事か忘れてしまったけど「虚数とゲーム」みたいなタイトルの記事だったと思う. 簡単に言っちゃえばiがπ/2分の回転をもたらすという記事.
    あれは高校生向けの記事だったので平面上のみの話題だった. 3DCGではいわゆるQuaternion(四元数)という概念に拡張する. 物理学も光の屈折や反射などCGと関係が深い.

    大学に入って同時にコンピュータ自体に興味持ってCG以外にも色々やっていた.
    と同時にゲームエンジンいじったり低レイヤレベルでOpenGLいじったり色々してた.

    と,そんな感じで今もCG関連の事をわちゃわちゃやってる.
    でもあんまりモデリングとかするのは得意じゃないかも.シェーダ書いたりする方が好きかもしれない.
